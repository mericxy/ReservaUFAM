# Histórias de Usuário de cada persona
- As personas do sistema possuem histórias do usuário que descrevem suas necessidades e interações dentro da aplicação.

Obs.: Só há as histórias de usuários, não há especificação aqui.
## Persona: Usuário
- H1 - Enquanto visitante, quero realizar meu cadastro para obter acesso às funcionalidades do sistema
- H2 - Enquanto usuário, quero fazer login para acessar o sistema
- H3 – Enquanto usuário, quero escolher entre tema claro ou escuro para usar o sistema de forma mais confortável.
- H4 — Enquanto usuário, quero acessar atalhos rápidos na página inicial para navegar para as principais funcionalidades.
- H5 — Enquanto usuário, quero solicitar uma reserva para utilizar um recurso.
- H6 — Enquanto usuário, quero visualizar e gerenciar minhas reservas para acompanhar e controlar minhas solicitações de uso.
- H7 - Enquanto usuário, quero acessar meu perfil para consultar minhas informações pessoais cadastradas.
- H8 — Enquanto usuário, quero editar minhas informações de perfil para manter meus dados sempre atualizados.
- H9 - Enquanto usuário, quero excluir minha conta permanentemente para remover os dados do sistema caso eu não deseje mais utilizá-lo.
- H10 — Enquanto usuário, quero recuperar minha senha por meio de um link enviado ao meu e-mail para restaurar meu acesso ao sistema.
- H17 – Enquanto usuário, quero ativar um modo de alto contraste para facilitar a leitura e navegação no sistema.
- H18 — Enquanto usuário, quero visualizar minhas próximas reservas na página inicial para acompanhar o que ainda está agendado.

## Persona: Administrador
- H11 — Enquanto administrador, quero visualizar um painel com o resumo das reservas e acessar rapidamente as reservas recentes para monitorar o status geral do sistema.
- H12 - Enquanto administrador, quero acessar atalhos para as páginas de gerenciamento de reservas, usuários e recursos para navegar facilmente entre as áreas administrativas do sistema.
- H13 — Enquanto administrador, quero gerenciar reservas pendentes, aprovadas e arquivadas para controlar o fluxo de solicitações de uso dos recursos.
- H14 – Enquanto administrador, quero gerenciar usuários em diferentes status para administrar o acesso ao sistema.
- H15 – Enquanto administrador, quero cadastrar novos recursos para manter o catálogo atualizado.
- H16 – Enquanto administrador, quero gerenciar e excluir recursos cadastrados para manter o sistema organizado.
